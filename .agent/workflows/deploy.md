# 위스키보다(WhiskyVoda) 배포 가이드 (Vercel)

이 프로젝트는 Next.js로 제작되어 **Vercel**을 통해 가장 쉽고 빠르게 배포할 수 있습니다.

### ⚠️ "vercel --prod" 경고가 뜨나요?

이 메시지는 현재 배포가 **'미리보기(Preview)'** 상태이며, 진짜 **'서비스(Production)'**로 올리려면 어떻게 해야 하는지 알려주는 친절한 가이드입니다. 오류가 아니니 걱정 마세요! 다음 두 방법 중 하나를 선택하시면 해결됩니다.

#### 방법 1: GitHub에 코드 푸시 (가장 추천!)
GitHub에 코드를 올리기만 하면 Vercel이 알아서 '진짜 서비스'로 배포해 줍니다.
```bash
git push origin main
```

#### 방법 2: 명령어로 직접 배포
GitHub을 거치지 않고 지금 바로 프로덕션으로 올리고 싶다면 다음 명령어를 입력하세요.
```bash
vercel --prod
```

> [!TIP]
> **방법 1**을 사용하면 앞으로 코드를 수정하고 `push` 할 때마다 자동으로 사이트가 업데이트되어 훨씬 편리합니다! ✨

> [!TIP]
> 배포 후 생성된 URL을 통해 전 세계 어디서든 위스키 매거진을 볼 수 있게 됩니다. ✨
